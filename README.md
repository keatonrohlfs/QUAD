# QUAD Marketplace
Live [here](http://ec2-18-208-129-43.compute-1.amazonaws.com/login)

## Usage
### To start
```
docker compose build
docker compose up
```
### To stop
Normal shutdown
```
docker compose down
```
Shutdown and clear databases
```
docker compose down -v
```
